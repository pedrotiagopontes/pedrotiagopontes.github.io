Halma Game

Client:

FEUP | Interdisciplinary Project

Halma is classic strategy board game, invented around 1883, which is played in a checkered board, typical divided in 16x16 squares.
In the first phase of this project we developed a prolog application that allowed multiple game modes such as: human vs human, human vs computer and computer vs computer. Furthermore, the game also supported three difficulty levels: easy (random algorithm), medium (greedy algorithm) and hard (minimax algorithm). Finally, the size of the board could also be changed to assure the flexibility and adaptability of the created algorithms.
In a second iteration, during the Lab of Graphical Interface Apps, our team developed a 3D game interface that used the previous prolog application as a independent module. The project was developed in C++ using OpenGL library and sockets to communicate with the prolog module.

Sapo Codebits | Feup

During an economic crisis like the one we are currently facing, managing your daily expenses is crucial, especially when you have a tight student budget.
This project was born during Sapo Codebits 2011 edition, as an android app aimed managed and record your incomes and outcomes. To each transaction the user could associate a tag (phone, food,...), a description, automatic location and a photo. In a second phase of development, our team created a dedicated REST service that allowed to maintain an online backup of all the transactions and multiple devices synchronized.
